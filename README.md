# Hastane-Ziyaret-Takip
Hastanede yatan hastalara yapılan ziyaretlerde, ziyaretçi bilgisi ve giriş-çıkış saatlerinin kaydını tutar.
Ziyaretçi giriş-çıkış saatlerinin düzenli ve kontrollü şekilde yapılmasını sağlamanın yanında ziyaretçi bilgilerini ve ziyaret süresini kaydederek güvenli takip sağlar.

Hasta gririşi ekranından hastanın kaydı yapılır.
Ziyaretçi giriş ekranından ziyaretçi kaydı yapılır.
Ziyaret giriş ekranında ise hangi ziyaretçinin hangi hastayı ziyaret ettiği kaydedilir.
